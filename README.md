# Traffic Keeper

> NAS 流量保活工具，自动下载大文件维持网络活跃

## 基本信息

| 项目 | 内容 |
|------|------|
| 版本 | 1.0.0 |
| 架构 | 全架构（x86 / ARM） |
| 安装空间 | 系统空间 |
| 运行端口 | 8080 |
| 发布者 | [545648820](https://github.com/w5456448820/) |

## 功能特性

- **自动保活**：自动下载大文件维持网络流量活跃，防止运营商因长期低流量而降速
- **Web 管理界面**：通过浏览器进行图形化配置、实时日志查看和链接管理
- **多下载源**：自动从 GitHub Release、镜像站等来源获取大文件链接
- **灵活配置**：支持限速、休眠时间、每日流量限额、下载超时等参数
- **可选单位**：时间支持 s/m/h，数据量支持 K/M/G/T（如 `30m`、`5G`）
- **飞牛原生**：以 FPK 原生应用形式运行，应用中心直接打开，无需 Docker

## 安装方式

### 通过 FnDepot 客户端安装（推荐）

1. 在飞牛 NAS 应用中心安装 [FnDepot](https://github.com/EWEDLCM/FnDepot)
2. 在 FnDepot 中添加本仓库地址：`https://github.com/w5456448820/traffic-keeper`
3. 找到 Traffic Keeper，点击安装

### 手动安装

1. 下载 [`traffic-keeper.fpk`](traffic-keeper/traffic-keeper.fpk)
2. 登录飞牛 NAS 管理后台，进入「应用中心」
3. 点击「手动安装」，选择下载的 `.fpk` 文件
4. 安装完成后在应用中心点击「打开」即可使用

## 使用说明

安装成功后，通过应用中心的「打开」按钮或浏览器访问 `http://<NAS_IP>:8080` 进入管理界面。

### 可配置参数

| 参数 | 说明 | 默认值 | 单位 |
|------|------|--------|------|
| MAX_SPEED | 最大下载速度 | 0（不限速） | K/M/G |
| MIN_SPEED | 最小保底速度 | 0（不限制） | K/M/G |
| SLEEP_MIN / SLEEP_MAX | 下载间隔休眠范围 | 5 / 15 | s/m/h |
| DAILY_LIMIT | 每日下载流量上限 | 0（不限制） | K/M/G/T |
| FETCH_INTERVAL | 链接抓取间隔 | 6h | s/m/h |
| LINK_CHECK_INTERVAL | 链接检测间隔 | 30m | s/m/h |
| WEB_PORT | Web 管理界面端口 | 8080 | - |
| DYNAMIC_SLEEP | 动态休眠（根据日进度调整） | false | - |

### Web 界面功能

- **配置管理**：浏览器内直接修改所有参数，保存即时生效
- **实时日志**：SSE 实时推送运行日志，支持关键词搜索
- **链接管理**：查看已抓取的下载链接，手动触发重新抓取
- **流量统计**：查看当日和历史下载流量统计

## 注意事项

- 本工具仅适合在你拥有管理权限的 NAS 和网络环境中使用
- 请合理设置限速、休眠时间和每日流量上限，避免影响正常网络使用
- Web 管理界面默认绑定 0.0.0.0:8080，在内网中可直接访问
- 如需限制访问，请通过飞牛 NAS 的防火墙或路由器 ACL 进行控制
