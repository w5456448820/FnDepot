# Traffic Keeper - 飞牛 NAS 流量保活工具

自动下载大文件维持网络活跃，防止运营商因长期低流量而降速。以飞牛 NAS 原生 FPK 应用形式运行，集成应用中心「打开」按钮。

## 功能

- 自动下载大文件保活网络
- Web 管理界面（配置 + 实时日志 + 链接管理）
- 多下载源自动切换（GitHub Release、镜像站）
- 可选单位：时间(s/m/h)、数据量(K/M/G/T)
- 每日流量限额、限速、动态休眠

## 安装

通过 FnDepot 添加本仓库源后安装，或手动下载 `traffic-keeper/traffic-keeper.fpk` 安装。

安装后通过应用中心「打开」按钮或访问 `http://<NAS_IP>:8080` 使用。
